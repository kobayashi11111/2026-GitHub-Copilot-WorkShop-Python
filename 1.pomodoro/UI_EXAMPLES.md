## ゲーミフィケーション要素の詳細

### 経験値(XP)システム
- **XP獲得**: 25分のポモドーロ完了で100XP
- **レベルアップ**: 500XPごとにレベルアップ
  - レベル1: 0 XP
  - レベル2: 500 XP
  - レベル3: 1000 XP
  - レベル4: 1500 XP
  - ...

### バッジシステム

#### ストリークバッジ（連続日数）
- 🔥 **3日連続**: 3日連続でポモドーロ実行
- 🔥🔥 **7日連続**: 1週間連続でポモドーロ実行
- 🔥🔥🔥 **30日連続**: 30日連続でポモドーロ実行
- 🔥🔥🔥🔥 **100日連続**: 100日連続でポモドーロ実行

#### 回数達成バッジ
- ⭐ **10回達成**: 累計10回のポモドーロ完了
- ⭐⭐ **25回達成**: 累計25回のポモドーロ完了
- ⭐⭐⭐ **50回達成**: 累計50回のポモドーロ完了
- ⭐⭐⭐⭐ **100回達成**: 累計100回のポモドーロ完了
- ⭐⭐⭐⭐⭐ **250回達成**: 累計250回のポモドーロ完了

### ストリーク表示
- **現在のストリーク**: 連続して実行している日数
- **最長ストリーク**: これまでの最長連続日数記録
- ポモドーロを1日でも逃すとストリークがリセット

### 統計機能
- **週間統計**: 過去7日間のデータ
- **月間統計**: 過去30日間のデータ
- **表示内容**:
  - 完了セッション数
  - 総作業時間（分・時間）
  - 1日平均セッション数
  - 日別完了数のグラフ（テキストベース）

## バッジ解放時の通知

新しいバッジを獲得すると、ポモドーロ完了画面で以下のように表示されます：

```
🎊 新しいバッジを獲得しました！
   🔥 3日連続達成！
   ⭐ 10回達成！
```

複数のバッジを同時に獲得した場合は、すべて表示されます。

## モチベーション向上の仕組み

1. **即座のフィードバック**: ポモドーロ完了時に獲得XPとレベルを即座に表示
2. **視覚的な進捗**: プログレスバーで次のレベルまでの進捗を可視化
3. **達成感の演出**: バッジ獲得時の特別な通知で達成感を高める
4. **長期目標**: ストリークとバッジで継続利用を促進
5. **成果の可視化**: 統計グラフで自分の成長を実感
