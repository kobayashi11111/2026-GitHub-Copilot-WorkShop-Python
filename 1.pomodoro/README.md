# 🍅 ポモドーロタイマー with ゲーミフィケーション

ゲーミフィケーション要素を持つポモドーロタイマーアプリケーションです。

## 主な機能

### コアタイマー機能
- ⏰ 25分の作業時間 + 5分の休憩時間
- カウントダウンタイマー
- セッション管理

### ゲーミフィケーション要素

#### 📊 経験値(XP)システム
- ポモドーロ完了ごとにXPを獲得（25分で100XP）
- レベルアップシステム（500XPごとにレベルアップ）
- 進捗バーで次のレベルまでの進捗を表示

#### 🏆 達成バッジシステム
**ストリークバッジ（連続日数）:**
- 🔥 3日連続達成
- 🔥🔥 7日連続達成
- 🔥🔥🔥 30日連続達成
- 🔥🔥🔥🔥 100日連続達成

**回数達成バッジ:**
- ⭐ 10回達成
- ⭐⭐ 25回達成
- ⭐⭐⭐ 50回達成
- ⭐⭐⭐⭐ 100回達成
- ⭐⭐⭐⭐⭐ 250回達成

#### 🔥 ストリーク表示
- 連続日数のカウント
- 最長ストリーク記録

#### 📈 統計機能
- 週間/月間統計
- 完了セッション数
- 総作業時間
- 1日平均セッション数
- 日別完了数のグラフ表示（テキストベース）

## 使い方

### 起動方法
```bash
cd 1.pomodoro
python app.py
```

### メニュー
1. **ポモドーロ開始**: 新しいポモドーロセッションを開始
2. **ステータス表示**: 現在のレベル、XP、ストリークなどを表示
3. **週間統計**: 過去7日間の統計を表示
4. **月間統計**: 過去30日間の統計を表示
5. **バッジ一覧**: 獲得したバッジを表示

### 操作方法
- ポモドーロ実行中は `Ctrl+C` で中断可能
- 作業完了後、休憩するかどうかを選択可能
- 各メニューからメインメニューに戻るには `Enter` キーを押す

## データ保存
- すべてのデータは `pomodoro_data.json` に自動保存されます
- セッション履歴、XP、バッジなどが永続化されます

## 技術仕様
- Python 3.x
- 標準ライブラリのみ使用（外部依存なし）
- JSONファイルでデータ永続化
- コマンドラインインターフェース

## 目的
ゲーミフィケーション要素がユーザーのモチベーション維持と継続利用に与える影響を測定します。

## スクリーンショット
実行すると以下のような画面が表示されます：
- メインメニュー
- タイマー画面
- ステータス表示
- 統計グラフ
- バッジコレクション
