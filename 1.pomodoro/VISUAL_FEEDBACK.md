# 視覚的フィードバック機能の説明

## 概要
ポモドーロタイマーに視覚的フィードバック機能を追加し、ユーザーの集中力と没入感を向上させます。

## 実装された機能

### 1. 円形プログレスバー
- Unicode文字（●、○、◉）を使用した円形の進捗表示
- 残り時間に応じて時計回りに進捗が減少
- 滑らかなアニメーション効果

```
        ●        
     ●○○●●●●     
   ○○○○○●●●●○○   
  ○○○○○○●●●○○○○  
  ○○○○○○◉◉◉○○○○  
 ○○○○○○○◉◉○○○○○○ 
```

### 2. 色のグラデーション変化
残り時間に応じて色が動的に変化します：

| 進捗率 | 色 | 意味 |
|-------|-----|------|
| 67-100% | 🔵 青 (Blue) | 十分な時間が残っています |
| 50-66% | 🔵 シアン (Cyan) | まだ余裕があります |
| 34-49% | 🟡 黄 (Yellow) | 半分を切りました |
| 20-33% | 🟠 オレンジ (Orange) | 時間が少なくなってきました |
| 0-19% | 🔴 赤 (Red) | 残りわずかです！ |

### 3. 背景エフェクト

#### パーティクルエフェクト
- 集中時間中に画面上部に表示
- 動く星（✦、✧、*、·、◦）がアニメーション
- リアルタイムで移動し、没入感を演出

```
  ·  ·                   *        *                     ·   
 ·        ✦                  ·         ·        ✦*          
                 *       ·                         ✦        
         *        ·        ·                ✦            ✧  
```

#### 波紋エフェクト
- 画面下部に表示される波のアニメーション
- サイン波を使用した自然な動き
- 波文字（~、≈、∼、⋍）で表現

```
~≈≈∼∼⋍⋍⋍⋍⋍⋍⋍∼≈≈~~~~~~~≈≈∼⋍⋍⋍⋍⋍⋍⋍∼∼≈≈~~~~~~~≈≈∼⋍⋍⋍⋍
```

### 4. 線形プログレスバー
- 円形プログレスバーの補助として表示
- 正確な進捗率をパーセンテージで表示
- 色はグラデーション変化と連動

```
進捗: [███░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░] 8.0%
```

### 5. 大きな時間表示
- ボックスで囲まれた見やすい時間表示
- ボールドフォントスタイル
- 色は残り時間に応じて変化

```
  ╔══════════════════════════╗
  ║         ⏰ 作業時間         ║
  ║                          ║
  ║          02:00           ║
  ║                          ║
  ╚══════════════════════════╝
```

## 使用方法

### 基本的な使い方

```python
from visual_feedback import VisualFeedback

# インスタンス作成
vf = VisualFeedback(width=60)

# 完全な視覚的表示を作成
display = vf.create_complete_display(
    remaining_seconds=900,    # 残り15分
    total_seconds=1500,       # 合計25分
    label="⏰ 作業時間",
    frame=0,                  # アニメーションフレーム番号
    show_particles=True       # パーティクル表示ON
)

print(display)
```

### アプリケーションでの使用

```python
from app import PomodoroTimer

# 視覚的フィードバックを有効にしてタイマー作成
timer = PomodoroTimer(
    work_duration=25,
    break_duration=5,
    use_visual_feedback=True  # 視覚的フィードバックを有効化
)

# メニューから「6. 視覚的フィードバック切替」で ON/OFF 可能
timer.show_menu()
```

## 技術仕様

### 使用技術
- **Python 3.x**: 標準ライブラリのみ使用
- **ANSI エスケープコード**: 色とカーソル制御
- **Unicode文字**: 円形表示と特殊文字
- **数学関数**: 円形プログレスバー計算（math.sqrt, math.atan2）

### モジュール構成

```
1.pomodoro/
├── visual_feedback.py          # 視覚的フィードバックモジュール
├── app.py                      # メインアプリ（統合済み）
└── tests/
    └── test_visual_feedback.py # テストコード（37テスト）
```

### クラス構造

```python
class VisualFeedback:
    """視覚的フィードバックを管理するクラス"""
    
    # 主要メソッド
    def get_color_for_progress(progress_percent)  # 色選択
    def create_circular_progress(progress_percent, radius)  # 円形バー
    def create_linear_progress_bar(progress_percent, width)  # 線形バー
    def create_wave_effect(frame, width)  # 波紋エフェクト
    def create_particles(frame, num_particles)  # パーティクル
    def format_time_display(mins, secs, progress, label)  # 時間表示
    def create_complete_display(...)  # 完全な表示
```

## 既知の制限事項

1. **ターミナル依存**: ANSI エスケープコードに対応したターミナルが必要
2. **画面サイズ**: 小さいターミナルでは表示が崩れる可能性
3. **カラー対応**: モノクロ端末では色の効果が見えない
