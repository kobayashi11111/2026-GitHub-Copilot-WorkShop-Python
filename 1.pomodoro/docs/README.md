## クイックスタート

### インストール

外部依存なし、Python 3.6以上が必要です。

```bash
cd 1.pomodoro
python app.py
```

### 基本的な使い方

1. アプリケーションを起動
2. メニューで `1` を選択してポモドーロ開始
3. 25分間作業
4. 完了後、XPとレベルを確認
5. 必要に応じて休憩（5分）

詳細は [CLI リファレンス](./cli-reference.md) を参照してください。

## 開発ガイド

### コードを理解する

1. **[アーキテクチャドキュメント](./architecture.md)** から読み始める
2. **[データモデル仕様](./data-models.md)** でデータ構造を把握
3. **[機能仕様](./features.md)** で各機能の実装詳細を確認
4. ソースコード（`app.py`, `gamification.py`）を読む

### 機能を追加する

1. **[機能仕様](./features.md)** の「拡張性」セクションを確認
2. 既存のアーキテクチャに従って実装
3. データモデルに変更が必要な場合は **[データモデル仕様](./data-models.md)** を更新
4. ユーザー向けドキュメント（**[CLI リファレンス](./cli-reference.md)**）を更新

### テストを書く

**[機能仕様](./features.md)** の「テスト」セクションを参照してテストケースを設計します。
