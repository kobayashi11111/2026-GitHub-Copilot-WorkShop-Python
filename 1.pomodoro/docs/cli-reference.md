# CLI リファレンス

## 概要

ポモドーロタイマーアプリケーションのコマンドラインインターフェース（CLI）は、対話型のメニューシステムで構成されています。

## 起動方法

### 基本的な起動

```bash
cd 1.pomodoro
python app.py
```

### 要件

- Python 3.x
- 外部ライブラリ不要（標準ライブラリのみ）

### 初回起動時

初回起動時、`pomodoro_data.json` ファイルが自動的に作成され、デフォルトデータで初期化されます。

## メインメニュー

アプリケーション起動時に表示されるメインメニュー：

```
==================================================
🍅 ポモドーロタイマー with ゲーミフィケーション
==================================================
1. ポモドーロ開始
2. ステータス表示
3. 週間統計
4. 月間統計
5. バッジ一覧
0. 終了
==================================================
```

## コマンド詳細

### 1. ポモドーロ開始

**機能**: 新しいポモドーロセッションを開始します。

**操作フロー**:
1. メインメニューで `1` を入力
2. 作業時間のカウントダウンが開始（デフォルト25分）
3. 完了時、獲得XP・レベル・新規バッジが表示
4. 休憩を取るか確認（y/n）
5. `y` を選択すると休憩タイマー開始（デフォルト5分）

**表示例**:
```
==================================================
🍅 ポモドーロタイマー開始！
==================================================
作業時間: 25分
Ctrl+C で中断

⏰ 作業時間: 24:35
```

**完了時の表示**:
```
==================================================
🎉 ポモドーロ完了！
==================================================
✨ 獲得XP: +100 (合計: 1500)
📊 レベル: 4
🔥 現在のストリーク: 3日

🎊 新しいバッジを獲得しました！
   🔥 3日連続達成！

==================================================

5分の休憩を取りますか？ (y/n): 
```

**中断方法**: 
- タイマー実行中に `Ctrl+C` を押す
- 中断した場合、ポモドーロとしてカウントされません

---

### 2. ステータス表示

**機能**: 現在のユーザーステータスを表示します。

**操作**: メインメニューで `2` を入力

**表示内容**:
- 現在のレベル
- 経験値（XP）と次のレベルまでの必要XP
- レベルアップ進捗バー
- 完了したポモドーロの総数
- 現在のストリーク（連続日数）
- 最長ストリーク記録
- 獲得バッジ数

**表示例**:
```
==================================================
📊 ステータス
==================================================
レベル: 4
経験値: 1500.0 / 2000
進捗: [███████████████░░░░░] 75.0%

合計ポモドーロ: 127回
現在のストリーク: 🔥 7日
最長ストリーク: 🔥 15日
獲得バッジ数: 5個
==================================================
```

**進捗バー**:
- 合計20文字
- 塗りつぶし: `█`
- 未塗りつぶし: `░`
- 5%ごとに1文字分進行

## ゲーミフィケーション要素

### 経験値（XP）システム

**獲得量**: 
- 25分のポモドーロ完了で100 XP
- 作業時間に比例: `(作業時間分 / 25) * 100`

**レベルアップ**:
- 500 XPごとにレベルアップ
- 計算式: `レベル = 1 + floor(総XP / 500)`

**レベルとXPの関係**:
```
レベル1: 0 XP ～ 499 XP
レベル2: 500 XP ～ 999 XP
レベル3: 1000 XP ～ 1499 XP
レベル4: 1500 XP ～ 1999 XP
...
```

### ストリーク（連続日数）

**カウント方法**:
- 毎日ポモドーロを1回以上完了するとストリークが継続
- 1日でも逃すとストリークがリセット
- 同日内に複数回完了してもストリークは1日分のみカウント

**表示**:
- 🔥アイコンとともに日数を表示
- ステータス画面と統計画面で確認可能

### バッジ

**獲得タイミング**: ポモドーロ完了時に条件を満たすと自動的に獲得

**通知**: 新規獲得時、完了画面で特別な通知が表示される

**種類**:
- **ストリークバッジ**: 連続実行日数に基づく（3日、7日、30日、100日）
- **回数達成バッジ**: 累計完了回数に基づく（10回、25回、50回、100回、250回）

## 画面制御

### クリア機能

各メニュー表示前に画面がクリアされます：
- Unix系OS（Linux, macOS）: `clear` コマンド
- Windows: `cls` コマンド

### 画面遷移

```
起動
 ↓
メインメニュー
 ↓
各機能画面（1-5）
 ↓
Enterキーで続行
 ↓
メインメニューに戻る
```

## データファイル

### 自動保存

ポモドーロ完了時に自動的に `pomodoro_data.json` へ保存されます。

**保存タイミング**:
- ポモドーロ完了直後
- XP・レベル・バッジ・ストリーク更新後

**ファイル場所**: アプリケーション実行ディレクトリ直下

### データのバックアップ

推奨: 定期的に `pomodoro_data.json` をバックアップ

```bash
# バックアップ例
cp pomodoro_data.json pomodoro_data.json.backup
```

## トラブルシューティング

### データファイルが破損した場合

1. アプリケーションを終了
2. `pomodoro_data.json` をリネームまたは削除
3. アプリケーションを再起動（新規データで初期化される）

```bash
# 破損したファイルをバックアップ
mv pomodoro_data.json pomodoro_data.json.broken

# アプリケーション再起動で新規作成
python app.py
```

### タイマーが表示されない場合

**原因**: ターミナルが標準出力のバッファリングを行っている

**対策**: 
- 実装では `flush=True` を使用しており、通常は問題ない
- 環境変数で無バッファモードを設定: `PYTHONUNBUFFERED=1 python app.py`

### 画面クリアが動作しない場合

**原因**: OSの識別ミス

**確認**:
```python
import os
print(os.name)  # 'posix' または 'nt'
```

**対策**: 手動でクリアコマンドを実行
```bash
# Unix系
clear

# Windows
cls
```

## カスタマイズ

### 作業時間・休憩時間の変更

現在のコードでは `PomodoroTimer` クラスのコンストラクタで設定可能：

```python
# app.py の main() 関数内
timer = PomodoroTimer(work_duration=25, break_duration=5)
```

**変更例（50分作業・10分休憩）**:
```python
timer = PomodoroTimer(work_duration=50, break_duration=10)
```

注意: XP計算は25分基準のため、作業時間を変更した場合はXP獲得量も変わります。

### データファイル名の変更

`GamificationManager` クラスのコンストラクタで指定可能：

```python
# app.py の PomodoroTimer.__init__() 内
self.gm = GamificationManager(data_file="custom_data.json")
```

## 使用例

### 典型的なワークフロー

```
1. アプリケーション起動
   $ python app.py

2. ステータス確認（メニューで「2」）
   → 現在のレベルとXPを確認

3. ポモドーロ開始（メニューで「1」）
   → 25分間作業
   → 完了後、XP獲得とバッジ通知を確認

4. 休憩（y を入力）
   → 5分間休憩

5. メインメニューに戻る（Enter）

6. 統計確認（メニューで「3」または「4」）
   → 今日の進捗と過去のデータを確認

7. 続けて作業する場合は 3. に戻る

8. 終了する場合はメニューで「0」
```

## キーボードショートカット

**タイマー実行中**:
- `Ctrl+C`: タイマーを中断

**メニュー画面**:
- `1-5`, `0`: メニュー選択
- `Enter`: 各画面からメインメニューに戻る

**休憩確認**:
- `y`: 休憩する
- `n`: 休憩をスキップ
